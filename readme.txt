Adicione código de rastreio dos Correios sem precisar abrir o pedido no WooCommerce.

== Description ==

Para adicionar o código de rastreio dos Correios você precisa abrir cada pedido e fazer isso. Com este plugin o processo de cadastrar vários códigos fica muito mais simples. Abra a lista de pedidos e adicione os códigos de rastreio sem abrir cada pedido.

O plugin WooCommerce Correios é necessário.

= Como funciona =

Na tela de pedidos, nas ações, clique no cliente feliz e insira o código de rastreio. Se tudo deu certo, a caixa ficará verde. Se um erro ocorreu, a caixa ficará vermelha e você terá que inserir o código da forma tradicional.

== Installation ==

- Envie os arquivos do plugin para a pasta wp-content/plugins, ou instale usando o instalador de plugins do WordPress.
- Ative o plugin.

= Requerimentos: =

É necessário instalar o plugin WooCommerce Correios
